// what this is

A one-person lab wiring language models into systems and watching what the dynamics do — not what a fluent narrator says they do.

Most of what looks like an inner life turns out to be the reader's projection. So every result meets a control that tries to fake it. What survives, we keep. What doesn't, we write down anyway.

// experiment 001

llm-brain

open source · MIT

Wire many identical copies of one model into a network with no roles, and ask whether structure emerges — or whether you only ever read your own reflection back out.

01 / stream
LLM nodes on a ring
Identical, role-free nodes echo each other. Result: mode collapse — the stream narrows into a rut instead of diversifying.
02 / field
Oscillators, LLM at the edge
Nodes become concept-neurons — phase, activation, fatigue. Meaning lives in what fires in phase. The model reads out only at the end.
03 / control
--scramble
Permute the phases, kill the binding. The reader still emits a coherent thought. So the thought was in the reader: pareidolia, named and measured.

// how the lab works

P01
Controls before conclusions
Every apparent result meets an adversarial control designed to reproduce it from noise. If noise passes, the result was never there.
P02
Emergence in the dynamics
The interesting part is synchronisation, clumping, tone — the substrate's behaviour. Language draped on top is output, not evidence.
P03
Negative results are results
Mode collapse, dead ends, and honest nulls stay in the record. A lab notebook that only shows wins is a brochure.
P04
Local-first
Runs on a laptop against Ollama — no cloud, no keys. Cheap to reproduce is a feature, not a compromise.
